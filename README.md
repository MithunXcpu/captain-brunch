A bill-splitting app that eliminates awkward math at the table. Enter the bill, add your crew, and everyone pays their share in seconds.

## Features

- **One-tap bill splitting** - Quick and easy bill division
- **Virtual payment cards** - Secure payments without sharing bank details
- **Share links** - Invite friends to split bills instantly
- **No more awkward math** - Automatic calculations for everyone

## Tech Stack

- **Framework**: Next.js 16 with App Router
- **Styling**: Tailwind CSS with custom design tokens
- **UI**: Custom components with warm espresso/burgundy/mustard theme
- **Fonts**: Geist (optimized via next/font)

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.
